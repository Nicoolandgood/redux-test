# Redux test project

A tiny react project to practice [Redux](https://redux.js.org/) and set up with [Vite.js](https://vitejs.dev/).

## Run the project
```sh
# Install the dependencies
yarn

# Run the development server
yarn dev
```
and go to [this page](http://localhost:3000).
